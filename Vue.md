花店管理系统
项目简介
这是一个基于 Vue 3 + Vite 开发的现代化花店管理系统前端项目。该项目采用最新的前端技术栈，提供了美观的用户界面和流畅的用户体验，旨在帮助花店实现数字化管理和运营。
技术栈
前端框架：Vue 3
构建工具：Vite
UI 组件库：Element Plus
图表库：ECharts
动画效果：Animate.css
状态管理：Vuex/Pinia
路由管理：Vue Router

用户管理

订单状态管理
订单统计和分析
数据可视化
销售数据统计
经营状况分析
数据报表导出
系统管理
系统配置
日志管理
数据备份
项目特点
采用 Vue 3 组合式 API，代码更简洁、更易维护
使用 Element Plus 组件库，提供统一的设计风格
集成 ECharts 图表库，实现数据可视化
响应式设计，支持多端适配
模块化架构，便于扩展和维护
